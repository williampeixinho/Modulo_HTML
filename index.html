Html 5

O que é?

HTML5 é a quinta versão da linguagem de marcação HTML (Hypertext Markup Language), que é usada para estruturar e apresentar conteúdo na World Wide Web (Internet).
Lançado em 2014, o HTML5 trouxe uma série de novos recursos, elementos e funcionalidades que possibilitam a criação de páginas web mais dinâmicas, interativas e acessíveis.
